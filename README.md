# Analog Clock

A responsive analog clock built with HTML, CSS, and JavaScript. The project renders a clean circular clock face with hour, minute, and second hands that update in real time using the user's local system time.

## Features

- Real-time analog clock movement
- Hour, minute, and second hands with distinct styling
- Minimal dark UI with a glass-style clock face
- Responsive centered layout
- Lightweight implementation with no external dependencies

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Getting Started

Open `index.html` in a browser to view the clock.

## Description

Analog Clock is a lightweight frontend project that demonstrates DOM manipulation, CSS transforms, and time-based JavaScript logic through an elegant real-time clock interface.
